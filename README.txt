TRANSPORTATION PENALTY TRACKER V2
=================================

1. Replace every current website file with the files in this folder.
2. Run database.sql in the Supabase SQL Editor.
3. Confirm pkaraffa@bridgeportedu.net exists in Authentication > Users.
4. Upload index.html, style.css, script.js, and supabase-config.js to GitHub Pages.
5. Press Ctrl+F5 after publishing.

The HTML uses ?v=2 on the files to prevent an old cached script from loading.
Open F12 > Console. You should see:
Transportation Penalty Tracker JavaScript loaded.
